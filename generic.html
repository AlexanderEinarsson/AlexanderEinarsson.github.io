Research Interests

My research interest mainly lies in using artificial intelligence to augment educators. With how ubiquitous AI has become, and how it has shown to improve so many fields, it is to me clear that we should attempt to work it into education. Some of my specific interests in this area includes how we can introduce big data analytics in the classroom, if it's possible to automate diagnosis of conceptual errors, and how we can create robust recommender system to help educators' decision making.
